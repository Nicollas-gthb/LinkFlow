# LinkFlow

Plataforma de gerenciamento de integrações e webhooks entre sistemas.

## Stack

- **Backend:** Java 21 + Spring Boot 3.5
- **Frontend:** React + Vite
- **Banco:** PostgreSQL
- **Infra:** Docker, Docker Compose, GitHub Actions

## Rodar local

```bash
docker compose up -d
```
